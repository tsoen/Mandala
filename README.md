# Projet Mandala

## Fonctionnalités réalisées

- Internationalisation : logiciel en anglais (de base), traduction française disponible et affichée automatiquement
- Choix du nombre de secteurs
- Choix de la taille du dessin
- Choix de l'épaisseur du trait
- Choix de la couleur du trait
- Afficher / masquer la grille
- Modifier l'opacité de la grille
- Fonction de dessin en mirroir
- Fonction de "nettoyer" le dessin : supprimer tous les traits dessinés
- Possibilité d'annuler les traits précedemment dessinés, et de "redessiner" une annulation (undo / redo)
- Fonction "teintes multiples / multicolore" : dupliquer le point dessiné en modifiant la teinte de la couleur selon l'angle (dans l'espace de couleurs HSV)
- Possibilité de sauvegarder le dessin dans un fichier image ("enregistrer dans le fichier en cours" et "enregistrer sous un nouveau fichier")
- Raccourcis claviers pour différentes actions (enregistrer, ouvrir, undo, redo..)

- Possibilité d'ouvrir un fichier image
- Possibilité de conserver uniquement le dessin sur un fond transparent (sauvegarde au format PNG).
- Vérification de l'enregistrement d'un fichier : le symbole "*" apparait après le nom du fichier ouvert lorsqu'il a subit des modifications non enregistrées.
    A la fermeture du logiciel, possibilité d'enregistrer le dessin pour ne pas perdre les modifications.
- Redimensionner une image existante
